 # Glorious Adventure

## Glorious Adventure - это наша командная проектная работа. Мы решили сделать небольшую браузерную игру и заодно получить опыт работы над одним проектом вместе.

В ходе разработки мы используем HTML, CSS и JavaScript, учимся работать с GitHub и постепенно добавляем новые идеи в игру. Сейчас проект ещё развивается, поэтому некоторые элементы могут меняться или дорабатываться.

### Что уже есть

* Главное меню
* Несколько игровых сцен
* Переходы между страницами
* Собственное оформление
* Размещение через GitHub Pages

### Зачем мы это делаем?
Нам хотелось не просто изучать теорию, а создать что-то своё. Этот проект помогает лучше понять веб-разработку, научиться работать в команде и получить опыт создания полноценного сайта с элементами игры.

### Что планируется дальше

* Новые локации
* Более интересный сюжет
* Улучшение дизайна
* Больше инттивности и игровых механик
